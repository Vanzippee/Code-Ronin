Why did I do it the way that I did it?
I used the mathematical principles of combinatorics. The best way to get to large number of combinations, is to use variables with the most posibilities.
Since the plates can use numbers and letters, the simplest pattern would result from each figure in the plate using all the possibilities. The assumption is that the simplest pattern will be the shortest pattern.
There are 35 possibilities using the digits 0-9 and the letters A-Z. The maximum number of plates can be determined using the formula #=35n, when n is the number of digits.

So the first step is to determine the least number of figures that will give enough combinations to provide unique plates for the entire population.

for (var n=0; Math.pow(35,(n+1))< pop; n++, z=n, x=Math.pow(35,n)) { }
This will determine the number of figures that will use all 35 possible figures.

The next step determines the least number of possibilities the last digit needs to have in order to provide enough combinations.

for (var i=1; Math.pow(35,z)*i < pop; i++, j=i, x=x*i) { }
Now we have a good plan to get the right number of plates with little waste.

The question is, can we get less waste. In order to determine if there is a better combination test the last two figures to determine if there is a combination for the last two digits that provides lest waste.

for (var i=1; i < 35; i++) { for (var k=1; k < 35; k++) { y=Math.pow(35, (z-1))*i*k; if (y > pop && (y-pop) < (x-pop)) { x=y; j=k; document.write(" < br > "); document.write("The second to last figure will use one of " +i); document.write(" figures."); } } }

I looked to see if testing more figures provided less waste, but I did not get less waste in my tests when I tested the third and fourth figures.

One problem I found was for populations less than 35. The first check would return no positive results so the whole check failed. In order to recover from this problem I added a final step to set the number of figures to one, and the number of possibilites from that set to the size of the population.
